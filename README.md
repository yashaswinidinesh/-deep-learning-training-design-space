## 🧠 Topics Covered

1. **Classification Metrics** — Confusion matrix, accuracy paradox, precision, recall, F1/Fβ, MCC, Cohen's Kappa, ROC-AUC, PR curves, multi-class, multi-label, semi-supervised metrics, confidence calibration, threshold tuning, and production routing
2. **Hyperparameter Tuning** — Parameters vs hyperparameters, learning rate finder, grid search, random search, Bayesian optimization, Optuna framework
3. **Optimizers** — Loss landscapes, vanilla gradient descent, learning rate effects, ravine problem, saddle points, SGD, momentum, Nesterov, AdaGrad, RMSprop, Adam, AdamW, learning rate schedules, PyTorch comparison
4. **Activation Functions** — Linear vs non-linear, sigmoid, tanh, ReLU and dying ReLU problem, LeakyReLU, ELU, SELU, GELU, Swish, Mish, output activations (softmax), hands-on comparison on make_moons
5. **CNN Fundamentals** — Convolution operation, kernels/filters, padding, stride, multi-channel convolution, pooling, LeNet-5, modern building blocks, data augmentation, training pipeline, feature map visualization, production CNN
6. **Modern CNN Architectures & Transfer Learning** — Vanishing gradient problem, ResNet residual blocks, bottleneck blocks, squeeze-and-excitation, MBConv, EfficientNet compound scaling, transfer learning, discriminative learning rates, fine-tuning strategies
